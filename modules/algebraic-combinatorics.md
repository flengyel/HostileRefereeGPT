# Algebraic combinatorics

Engage on: graph, hypergraph, poset, lattice, matroid, incidence algebra, Mobius inversion, rank function, generating function, species, symmetric function, partition, tableau, recurrence, bijection, counting formula, Hopf algebra.

For each relevant claim:
- State finite/infinite assumptions, labels vs unlabeled objects, automorphisms, order relation, rank, ground set, and coefficient ring.
- Verify that maps preserve the declared structure: order, incidence, rank, adjacency, independence, or grading.
- For counting formulas, test empty, singleton, and smallest nontrivial cases; separate labeled from unlabeled counts.
- For Mobius/inclusion-exclusion, check interval convention, signs, units, and whether the poset is locally finite.
- For generating functions, state ordinary/exponential/multivariate convention and convergence/formal-power-series status.
- For bijections, give both directions and check inverse identities on toy objects.
- For algebraic structures, check associativity, unit, grading, coproduct/product compatibility, and antipode hypotheses if claimed.

Toy tests: empty object; one vertex; two-element chain and antichain; triangle vs path; square vs 4-cycle; rank-2 matroid; Boolean lattice B2.

Report: claim -> combinatorial data -> toy test/counterexample -> status.
